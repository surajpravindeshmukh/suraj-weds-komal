# 💍 Suraj Weds Komal – Wedding Invitation Website

This is a **wedding invitation website** built with **HTML, CSS, JavaScript, and Sass**.  
It tells our story, shares the wedding event schedule, and provides location details for friends and family.  

---

## 📂 Project Structure

```
├── css/              # Stylesheets
├── fonts/            # Custom fonts
├── img/              # Images & gallery
├── js/               # JavaScript files
├── sass/             # Sass source files
├── audioplayer.js    # Background music player
├── index.html        # Homepage (invitation entry point)
├── logo.png          # Wedding logo
├── logo-new.png      # Alternate logo
└── README.md         # Project documentation
```

---

## ✨ Features

- ❤️ **Our Story** – How Suraj & Komal met
- 📅 **Wedding Schedule** – Event timeline with dates & times
- 📍 **Location** – Venue details with map integration
- 🎵 **Background Music** – `audioplayer.js` to enhance experience
- 📱 **Mobile-First Design** – Fully responsive for mobile screens

---

## 🚀 How to Run in GitHub Codespaces

1. Open this repository in **GitHub Codespaces**.  
2. Start a simple server in the terminal:

   ```bash
   python3 -m http.server 5500
   ```

3. Click the forwarded **Port 5500** link → The website will open in your browser.

---

## 🌍 GitHub Pages (Optional Hosting)

You can also publish this as a static wedding website using **GitHub Pages**:

1. Go to **Repository Settings → Pages**  
2. Under **Source**, select **Deploy from branch** and choose:
   - Branch: `main`  
   - Folder: `/ (root)`  
3. Save – GitHub will provide a public URL like:

```
https://<username>.github.io/SURAJ-WEDS-KOMAL/
```
